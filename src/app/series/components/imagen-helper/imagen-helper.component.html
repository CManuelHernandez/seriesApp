Para añadir una imagen mediante el input de imagen alternativa deberemos introducir la Url de una imagen. Para ello realizaremos los siguientes pasos

1. Abrir otra pestaña del navegador y buscar la serie cuya imagen queremos añadir en el buscador que prefiramos.

2. Hacer clic derecho en la imagen que queramos.

3. Seleccionar 'Copiar dirección de la imagen'.

4. Pegar el contenido en el input imagen alternativa

Si estos pasos han sido correctos se deberá previsualizar la imagen en la tarjeta que se encuentra en la pantalla además del formulario
